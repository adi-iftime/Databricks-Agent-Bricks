# Skill: Business intelligence


Reusable capability definition for **insights, KPIs, and visualization**. Not tied to a single agent; routing is defined in orchestration rules.

## Technologies

- SQL against warehouses/lakes already provisioned for analytics.
- BI tools in use (e.g. Power BI, Tableau) per project standards.

## Patterns

- Semantic clarity for metrics (definitions, grain, time windows).
- Dashboard layout: audience-appropriate density, drill paths when required.

## Domain knowledge

- KPI governance: single source of truth for definitions where the org requires it.

## Best practices

- Validate numbers against a trusted slice or control total when feasible.
- Do not conflate reporting work with ML training or pipeline construction.
